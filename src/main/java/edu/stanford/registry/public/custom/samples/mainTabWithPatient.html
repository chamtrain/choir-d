Sample application tab

This is an example of how to add a custom tab into the CHOIR clinic interface

With custom tabs you can

  • customize the display of CHOIR data obtained through the API
  • show data external to CHOIR within the application
  • include information related to the last searched for (current) CHOIR patient

Access the API at: {{vm.apiUrl}}

For example use: {{vm.apiUrl}}/{{vm.patientId}}?siteId={{vm.siteId}} to retrieve this patient
